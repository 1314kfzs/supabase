# PortfolioBlog 功能说明文档

## 项目概述

PortfolioBlog 是一个现代化的个人作品集和技术博客平台，旨在帮助开发者展示技术能力和分享学习经验。

## 核心功能

### 1. 首页 (/)
- **个人简介展示区**: 展示开发者基本信息和个人介绍
- **特色作品轮播**: 精选3-5个重要项目进行重点展示
- **最新博客文章预览**: 显示最新发布的3篇技术博客
- **技能标签云**: 以标签云形式展示技术栈和技能
- **社交链接入口**: 快速访问GitHub、LinkedIn等社交平台

### 2. 作品集页面 (/projects)
- **作品网格布局**: 采用响应式网格布局展示所有项目
- **技术栈筛选功能**: 按技术栈（React、TypeScript等）筛选项目
- **关键词搜索**: 实时搜索项目标题和描述
- **分页加载**: 每页显示9个项目，支持分页浏览
- **项目详情模态框**: 点击项目查看详细信息，包括：
  - 项目封面图片
  - 详细项目描述
  - 技术栈标签
  - 项目链接（GitHub、演示地址等）
  - 项目状态和时间信息

### 3. 博客页面 (/blog)
- **博客文章列表**: 按时间倒序展示所有已发布文章
- **文章分类筛选**: 按技术分类（React、Node.js等）筛选文章
- **关键词搜索**: 搜索文章标题和内容
- **分页显示**: 每页显示6篇文章
- **文章详情页**: 完整的博客文章阅读体验，包括：
  - 文章标题和元信息（发布时间、阅读时长）
  - 文章标签分类
  - Markdown格式内容渲染
  - 代码高亮显示
  - 相关文章推荐

### 4. 用户认证系统
- **用户注册**: 新用户注册账户
- **用户登录**: 现有用户登录系统
- **会话管理**: 自动保持登录状态
- **安全退出**: 安全退出登录

## 技术特性

### 前端技术栈
- **React 18**: 现代化React框架
- **TypeScript**: 类型安全的JavaScript
- **Tailwind CSS**: 实用优先的CSS框架
- **React Router**: 单页应用路由管理
- **TanStack Query**: 数据获取和状态管理
- **Zustand**: 轻量级状态管理

### 后端服务
- **Supabase**: 后端即服务（BaaS）
  - 用户认证管理
  - 实时数据库
  - 文件存储
  - 安全策略

### 部署和性能
- **Netlify**: 静态站点部署
- **响应式设计**: 支持移动端和桌面端
- **性能优化**: 代码分割和懒加载

## 数据模型

### 用户信息 (profiles)
- 基本信息：用户名、头像、个人简介
- 联系信息：邮箱、GitHub、LinkedIn等
- 技能标签：技术栈和技能列表

### 项目信息 (projects)
- 项目基本信息：标题、描述、封面图
- 技术信息：使用的技术栈
- 链接信息：GitHub地址、演示地址等
- 时间信息：开始时间、完成时间
- 状态信息：项目状态（进行中/已完成）

### 博客文章 (blog_posts)
- 文章内容：标题、内容、摘要
- 元信息：发布时间、阅读时长、浏览量
- 分类信息：标签和分类
- 发布状态：草稿/已发布

## 安全特性

### 数据安全
- **行级安全策略**: 数据库级别的访问控制
- **用户权限管理**: 用户只能管理自己的数据
- **公开数据访问**: 作品和博客文章对所有人可见

### 认证安全
- **安全的密码存储**: 使用Supabase的安全认证
- **会话管理**: 安全的会话保持机制
